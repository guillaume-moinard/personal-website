But first, what is a graph ?

A graph is a pair G = (V, E), where V is a set whose elements are called vertices, and E is a set of paired vertices, whose elements are called edges.

Shortest path in a link stream

Link streams are a formalism to analyse and represent dynamic graphs. They are a generalization of static graphs and exhibits commun properties, such as shortest path. They are a very usefull tool in our studies.
